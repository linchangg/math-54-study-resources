MUC — Basic Examples

No overlap. Identify the type of \(f(t)\), write the corresponding guess, differentiate, plug in, match coefficients.

Lecture · B4.4.11 \(y'' + y = 2^x\) Rewrite as \(e^{kx}\) ▾
1
Rewrite f(x)
Use the identity \(a^x = e^{x \ln a}\):
\[ 2^x = e^{x \ln 2} \]
Now it's a plain exponential with \(k = \ln 2\). MUC is viable.
2
Complementary solution — check overlap
\[ r^2 + 1 = 0 \;\Rightarrow\; r = \pm i \;\Rightarrow\; y_c = c_1 \cos x + c_2 \sin x \]
No \(e^{x\ln 2}\) in \(y_c\). No overlap.
3
Guess & differentiate — color tracks the single unknown
gold = unknown \(A\)
\[\begin{aligned} y_p &= {\color{#e8b86d}{A}}\, e^{x\ln 2}\\[4pt] y_p' &= {\color{#e8b86d}{A}}(\ln 2)\, e^{x\ln 2}\\[4pt] y_p'' &= {\color{#e8b86d}{A}}(\ln 2)^2\, e^{x\ln 2} \end{aligned}\]
4
Substitute & equate coefficients
\[ {\color{#e8b86d}{A}}(\ln 2)^2\, e^{x\ln 2} + {\color{#e8b86d}{A}}\, e^{x\ln 2} = e^{x\ln 2} \]
Divide both sides by \(e^{x\ln 2} \neq 0\):
\[ {\color{#e8b86d}{A}}\bigl[(\ln 2)^2 + 1\bigr] = 1 \;\Longrightarrow\; {\color{#e8b86d}{A}} = \frac{1}{(\ln 2)^2 + 1} \]
General solution
\[ y = c_1 \cos x + c_2 \sin x + \frac{2^x}{(\ln 2)^2 + 1} \]
Lecture · B4.4.3 \(2y'' - 6y' + y = e^{-4x}\sin x\) Product guess ▾
1
Identify and check viability
\(f(x) = e^{-4x}\sin x\) is exponential \(\times\) trig — both MUC-viable, so the product is too. It would fail if it were \(e^{4x}/\sin x = e^{4x}\csc x\) since \(\csc x\) is not a viable type.
2
Complementary solution — check overlap
\[ 2r^2 - 6r + 1 = 0 \;\Rightarrow\; r = \frac{6 \pm \sqrt{28}}{4} \quad\text{(real irrational)} \]
\(y_c\) contains only real exponentials — no \(e^{-4x}\sin x\). No overlap.
3
Form the guess — product of individual guesses
gold = \(A\)
teal = \(B\)
red = shared \(e^{-4x}\)
\[ y_p = {\color{#c47b7b}{e^{-4x}}}\!\bigl({\color{#e8b86d}{A}}\cos x + {\color{#7eb8a4}{B}}\sin x\bigr) \]
No extra constant \(C\) out front. Writing \(C \cdot e^{-4x}(A\cos x + B\sin x)\) makes \(CA\) just one constant — two symbols for the same unknown. Redundant and makes the algebra longer for no reason.
4
Differentiate & plug in
Each derivative uses the product rule, pulling a factor of \(-4\) from \(e^{-4x}\) and cycling \(\cos x \leftrightarrow \sin x\). After substituting into \(2y'' - 6y' + y\), group by \({\color{#c47b7b}{e^{-4x}}}\cos x\) and \({\color{#c47b7b}{e^{-4x}}}\sin x\) terms separately and set each coefficient equal to the right-hand side.
Form of particular solution
\[ y_p = {\color{#c47b7b}{e^{-4x}}}\bigl({\color{#e8b86d}{A}}\cos x + {\color{#7eb8a4}{B}}\sin x\bigr) \]
Lecture · B4.4.7 \(8z'' - 2z' = 3x^{100}e^{4x}\cos(25x)\) Triple product ▾
1
Three factors — all viable
gold = polynomial \(x^{100}\)
red = exponential \(e^{4x}\)
blue = trig \(\cos(25x)\)
\[ f(x) = 3{\color{#e8b86d}{x^{100}}}\cdot{\color{#c47b7b}{e^{4x}}}\cdot{\color{#8b9eca}{\cos(25x)}} \]
2
Guess = product of three individual guesses
\[ z_p = {\color{#e8b86d}{q(x)}}\cdot{\color{#c47b7b}{e^{4x}}}\cdot\bigl({\color{#8b9eca}{A\cos(25x) + B\sin(25x)}}\bigr) \]
where \({\color{#e8b86d}{q(x)}}\) is a general degree-100 polynomial with 101 unknown coefficients.
Form of z_p
\[ z_p = {\color{#e8b86d}{q(x)}}\;{\color{#c47b7b}{e^{4x}}}\bigl[{\color{#8b9eca}{A\cos(25x) + B\sin(25x)}}\bigr], \quad \deg(q)=100 \]
MUC — Overlap Examples

When your guess is already in \(y_c\) it contributes zero on the left side. Multiply by \(t\) (or \(t^2\) for a double root) to fix it.

Lecture · Overlap Demo \(y'' + y = \cos t\) Why the naive guess fails ▾
1
Complementary solution
\[ r^2 + 1 = 0 \;\Rightarrow\; r = \pm i \;\Rightarrow\; {\color{#c47b7b}{y_c = c_1\cos t + c_2\sin t}} \]
2
Naive guess produces zero — here's exactly why
Normal guess: \(y_p = {\color{#e8b86d}{A}}\cos t + {\color{#7eb8a4}{B}}\sin t\). Differentiate twice:
\[\begin{aligned} y_p &= {\color{#e8b86d}{A}}\cos t + {\color{#7eb8a4}{B}}\sin t \\[4pt] y_p'' &= -{\color{#e8b86d}{A}}\cos t - {\color{#7eb8a4}{B}}\sin t \end{aligned}\]
\[ y_p'' + y_p = (-{\color{#e8b86d}{A}}+{\color{#e8b86d}{A}})\cos t + (-{\color{#7eb8a4}{B}}+{\color{#7eb8a4}{B}})\sin t = {\color{#c47b7b}{0}} \]
⚠ The guess satisfies the homogeneous equation, so it always gives \(0\) on the left — never \(\cos t\).

Overlap test: ratio \(= \cos t / \cos t = 1\), a polynomial → overlap confirmed.
3
Fix: multiply the whole guess by \(t\)
\[ y_p = t\bigl({\color{#e8b86d}{A}}\cos t + {\color{#7eb8a4}{B}}\sin t\bigr) \]
Differentiate twice by product rule:
\[\begin{aligned} y_p' &= {\color{#e8b86d}{A}}\cos t + {\color{#7eb8a4}{B}}\sin t + t(-{\color{#e8b86d}{A}}\sin t + {\color{#7eb8a4}{B}}\cos t)\\[4pt] y_p'' &= -2{\color{#e8b86d}{A}}\sin t + 2{\color{#7eb8a4}{B}}\cos t + t(-{\color{#e8b86d}{A}}\cos t - {\color{#7eb8a4}{B}}\sin t) \end{aligned}\]
4
Substitute — \(t\) terms cancel, only constants survive
\[\begin{aligned} y_p'' + y_p &= -2{\color{#e8b86d}{A}}\sin t + 2{\color{#7eb8a4}{B}}\cos t\\ &\quad + t\underbrace{\bigl(-{\color{#e8b86d}{A}}\cos t - {\color{#7eb8a4}{B}}\sin t + {\color{#e8b86d}{A}}\cos t + {\color{#7eb8a4}{B}}\sin t\bigr)}_{=\;0}\\[4pt] &= -2{\color{#e8b86d}{A}}\sin t + 2{\color{#7eb8a4}{B}}\cos t \;=\; \cos t \end{aligned}\]
\[ 2{\color{#7eb8a4}{B}} = 1 \;\Rightarrow\; {\color{#7eb8a4}{B = \tfrac{1}{2}}} \qquad\qquad -2{\color{#e8b86d}{A}} = 0 \;\Rightarrow\; {\color{#e8b86d}{A = 0}} \]
General solution
\[ y = c_1\cos t + c_2\sin t + \frac{t}{2}\sin t \]
Lecture · Overlap \(y'' + y = t\cos t\) Product \(\times\) overlap combined ▾
1
y_c is the same
\[ {\color{#c47b7b}{y_c = c_1\cos t + c_2\sin t}} \]
2
What the naive guess would be — and why it fails
\(f(t) = t\cos t\) is polynomial \(\times\) trig, so normally guess a degree-1 polynomial times both trig functions:
\[ y_p^{\text{naive}} = ({\color{#e8b86d}{A}}t + {\color{#c47b7b}{B}})\cos t + ({\color{#7eb8a4}{C}}t + {\color{#c47b7b}{D}})\sin t \]
But \({\color{#c47b7b}{B\cos t}}\) and \({\color{#c47b7b}{D\sin t}}\) are in \(y_c\). They contribute zero. Discard them.
3
Correct guess — the whole naive guess is multiplied by \(t\)
\(A, C\) multiply \(t^2\)
\(B, D\) multiply \(t\)
\[ y_p = ({\color{#e8b86d}{A}}t^2 + {\color{#7eb8a4}{B}}t)\cos t + ({\color{#e8b86d}{C}}t^2 + {\color{#7eb8a4}{D}}t)\sin t \]
Do not write \((At^2 + Bt + E)\cos t\). The \(E\cos t\) term is a homogeneous solution — it contributes nothing and adds unnecessary unknowns.
Correct form — solving gives \(A=0,\,B=1,\,C=1,\,D=0\)
\[ y_p = t\cos t + t^2\sin t \]
Lecture · Double Overlap \(y'' - 2y' + y = e^t\) Double root \(\Rightarrow\) multiply by \(t^2\) ▾
1
Complementary solution — double root
\[ r^2 - 2r + 1 = (r-1)^2 = 0 \;\Rightarrow\; r = 1,\;1 \]
\[ {\color{#c47b7b}{y_c = c_1 e^t + c_2\, t e^t}} \]
2
Both \(e^t\) and \(te^t\) are in \(y_c\)
Normal guess \({\color{#c47b7b}{Ae^t}}\) — in \(y_c\). Multiply by \(t\): \({\color{#c47b7b}{Ate^t}}\) — also in \(y_c\). Double overlap → multiply by \(t^2\).
⚠ General rule: if the overlapping root has multiplicity \(m\), multiply the standard guess by \(t^m\).
3
Correct guess and its derivatives
\[\begin{aligned} y_p &= {\color{#e8b86d}{A}}t^2 e^t\\[4pt] y_p' &= {\color{#e8b86d}{A}}(t^2 + 2t)\,e^t\\[4pt] y_p'' &= {\color{#e8b86d}{A}}(t^2 + 4t + 2)\,e^t \end{aligned}\]
4
Substitute — \(t^2\) and \(t\) terms cancel, only the constant survives
\[\begin{aligned} &{\color{#e8b86d}{A}}(t^2+4t+2)e^t - 2{\color{#e8b86d}{A}}(t^2+2t)e^t + {\color{#e8b86d}{A}}t^2 e^t\\[4pt] &= {\color{#e8b86d}{A}}e^t\bigl[\underbrace{(1-2+1)}_{=\,0}t^2 + \underbrace{(4-4)}_{=\,0}t + 2\bigr]\\[4pt] &= 2{\color{#e8b86d}{A}}e^t = e^t \end{aligned}\]
\[ {\color{#e8b86d}{A = \tfrac{1}{2}}} \]
General solution
\[ y = c_1 e^t + c_2\,t e^t + \tfrac{1}{2}t^2 e^t \]
Lecture · Overlap Nuance \(y'' + 9y = e^t \sin(3t)\) Shared trig but NO overlap ▾
1
Complementary solution
\[ r^2 + 9 = 0 \;\Rightarrow\; r = \pm 3i \;\Rightarrow\; {\color{#c47b7b}{y_c = c_1\cos(3t) + c_2\sin(3t)}} \]
2
Apply the ratio test
Both \(f(t)\) and \(y_c\) contain \(\sin(3t)\). Compute the ratio:
\[ \frac{e^t\sin(3t)}{\sin(3t)} = e^t \]
\(e^t\) is not a polynomial → no overlap.
⚠ Compare: \(t\cos t\,/\,\cos t = t\) (polynomial → overlap). \(\;e^t\cos t\,/\,\cos t = e^t\) (not polynomial → no overlap). The exponential factor changes everything.
3
Standard guess — no \(t\) factor needed
\[ y_p = {\color{#e8b86d}{A}}e^t\cos(3t) + {\color{#7eb8a4}{B}}e^t\sin(3t) \]
Form of y_p
\[ y_p = {\color{#e8b86d}{A}}e^t\cos(3t) + {\color{#7eb8a4}{B}}e^t\sin(3t) \quad\text{(no }t\text{ factor needed)} \]

Lecture · B4.5.17 \(y'' - 2y' - 3y = 3t^2 - 5\) Full polynomial solve with three unknowns ▾
1
Complementary solution
\[ r^2 - 2r - 3 = (r-3)(r+1) = 0 \;\Rightarrow\; r = -1,\;3 \]
\[ y_c = c_1 e^{-t} + c_2 e^{3t} \]
No polynomial terms in \(y_c\) → no overlap with \(f(t) = 3t^2-5\).
2
Guess — degree matches \(f\)
gold = \(A\) (coeff of \(t^2\))
teal = \(B\) (coeff of \(t\))
purple = \(C\) (constant)
\[ y_p = {\color{#e8b86d}{A}}t^2 + {\color{#7eb8a4}{B}}t + {\color{#b48fd4}{C}} \]
\[ y_p' = 2{\color{#e8b86d}{A}}t + {\color{#7eb8a4}{B}}, \qquad y_p'' = 2{\color{#e8b86d}{A}} \]
3
Substitute and expand
\[ 2{\color{#e8b86d}{A}} - 2(2{\color{#e8b86d}{A}}t + {\color{#7eb8a4}{B}}) - 3({\color{#e8b86d}{A}}t^2 + {\color{#7eb8a4}{B}}t + {\color{#b48fd4}{C}}) = 3t^2 - 5 \]
\[ -3{\color{#e8b86d}{A}}t^2 + (-4{\color{#e8b86d}{A}}-3{\color{#7eb8a4}{B}})t + (2{\color{#e8b86d}{A}}-2{\color{#7eb8a4}{B}}-3{\color{#b48fd4}{C}}) = 3t^2 + 0\cdot t - 5 \]
4
Equate coefficients — one equation per power of \(t\)
\[\begin{alignedat}{3} t^2\!:\quad & -3{\color{#e8b86d}{A}} = 3 &\quad&\Rightarrow\quad & {\color{#e8b86d}{A}} &= -1\\[4pt] t^1\!:\quad & -4{\color{#e8b86d}{A}}-3{\color{#7eb8a4}{B}} = 0 &\quad&\Rightarrow\quad & {\color{#7eb8a4}{B}} &= \tfrac{4}{3}\\[4pt] t^0\!:\quad & 2{\color{#e8b86d}{A}}-2{\color{#7eb8a4}{B}}-3{\color{#b48fd4}{C}} = -5 &\quad&\Rightarrow\quad & {\color{#b48fd4}{C}} &= \tfrac{1}{9} \end{alignedat}\]
General solution
\[ y(t) = c_1 e^{-t} + c_2 e^{3t} \underbrace{- t^2 + \tfrac{4}{3}t + \tfrac{1}{9}}_{y_p} \]
Lecture · B4.5.21 \(y'' + 2y' + 2y = e^{-t}\cos t\) Overlap with complex exponential ▾
1
Complementary solution
\[ r^2+2r+2=0 \;\Rightarrow\; (r+1)^2+1=0 \;\Rightarrow\; r=-1\pm i \]
\[ {\color{#c47b7b}{y_c = c_1 e^{-t}\cos t + c_2 e^{-t}\sin t}} \]
2
Overlap check
Normal guess: \({\color{#c47b7b}{Ae^{-t}\cos t + Be^{-t}\sin t}}\). These are exactly in \(y_c\).
\[ \frac{e^{-t}\cos t}{e^{-t}\cos t} = 1 \quad\text{(polynomial)}\;\Rightarrow\;\textbf{overlap} \]
Root \(r=-1+i\) is not repeated, so this is a single overlap → multiply by \(t\) once.
3
Corrected guess
\[ y_p = t\,{\color{#c47b7b}{e^{-t}}}\bigl({\color{#e8b86d}{A}}\cos t + {\color{#7eb8a4}{B}}\sin t\bigr) \]
Form of y_p
\[ y_p = t\, e^{-t}\bigl({\color{#e8b86d}{A}}\cos t + {\color{#7eb8a4}{B}}\sin t\bigr) \]
Variation of Parameters

When \(f(t)\) is not MUC-viable. Color tracks each function \(y_1,\,y_2,\,u_1,\,u_2\) and the Wronskian \(W\) through every step.

What is \(a\) in the formula \(u_1' = -f\,y_2\,/\,(a\,W)\)?
The general equation is \({\color{#8b9eca}{a}}y'' + by' + cy = f(t)\). When you divide through to get standard form \(y'' + \cdots = f(t)/{\color{#8b9eca}{a}}\), that leading coefficient \({\color{#8b9eca}{a}}\) ends up in the denominator of both VOP formulas. In most textbook problems \({\color{#8b9eca}{a=1}}\) so it disappears — but if you have e.g.\ \(2y'' + \ldots\), then \({\color{#8b9eca}{a=2}}\) and you must divide by it.
Lecture · B4.6.1 \(y'' + 4y = \tan(2t)\) VOP with trig integrals ▾
1
Why not MUC?
\(f(t) = \tan(2t)\) is not a polynomial, exponential, or sin/cos — MUC fails. Use VOP. Note \({\color{#8b9eca}{a = 1}}\) here so the \(a\) in the denominator disappears.
2
Find \(y_1,\,y_2\) and compute the Wronskian
\(y_1 = \cos(2t)\)
\(y_2 = \sin(2t)\)
\(W\) = Wronskian
\[ r^2+4=0 \;\Rightarrow\; r=\pm 2i \;\Rightarrow\; {\color{#e8b86d}{y_1=\cos(2t)}},\quad {\color{#7eb8a4}{y_2=\sin(2t)}} \]
\[ {\color{#8b9eca}{W}} = \begin{vmatrix}{\color{#e8b86d}{\cos(2t)}} & {\color{#7eb8a4}{\sin(2t)}}\\ -2{\color{#e8b86d}{\sin(2t)}} & 2{\color{#7eb8a4}{\cos(2t)}}\end{vmatrix} = 2\cos^2(2t)+2\sin^2(2t) = {\color{#8b9eca}{2}} \]
3
Cramer's Rule — solve for \(u_1'\) and \(u_2'\)
The two formulas come from solving the \(2{\times}2\) linear system by Cramer's Rule. Watch how \({\color{#e8b86d}{y_1}}\), \({\color{#7eb8a4}{y_2}}\), \(f\), and \({\color{#8b9eca}{W}}\) slot in:
\[ u_1' = \frac{-{\color{#7eb8a4}{y_2}}\cdot f}{{\color{#8b9eca}{W}}} = \frac{-{\color{#7eb8a4}{\sin(2t)}}\cdot\tan(2t)}{{\color{#8b9eca}{2}}} = \frac{-\sin^2(2t)}{2\cos(2t)} \]
\[ u_2' = \frac{{\color{#e8b86d}{y_1}}\cdot f}{{\color{#8b9eca}{W}}} = \frac{{\color{#e8b86d}{\cos(2t)}}\cdot\tan(2t)}{{\color{#8b9eca}{2}}} = \frac{\sin(2t)}{2} \]
4
Integrate \(u_1'\) using \(\sin^2 = 1 - \cos^2\)
\[\begin{aligned} u_1 &= -\tfrac{1}{2}\int\frac{\sin^2(2t)}{\cos(2t)}\,dt = -\tfrac{1}{2}\int\frac{1-\cos^2(2t)}{\cos(2t)}\,dt\\[6pt] &= -\tfrac{1}{2}\int\bigl[\sec(2t)-\cos(2t)\bigr]\,dt\\[6pt] &= -\tfrac{1}{4}\bigl[\ln|\sec(2t)+\tan(2t)| - \sin(2t)\bigr] \end{aligned}\]
5
Integrate \(u_2'\)
\[ u_2 = \tfrac{1}{2}\int\sin(2t)\,dt = \frac{-\cos(2t)}{4} \]
6
Assemble \(y_p = u_1 {\color{#e8b86d}{y_1}} + u_2 {\color{#7eb8a4}{y_2}}\)
\[\begin{aligned} y_p &= \Bigl(-\tfrac{1}{4}\ln|\sec(2t)+\tan(2t)| + \tfrac{\sin(2t)}{4}\Bigr){\color{#e8b86d}{\cos(2t)}}\\ &\quad + \frac{-{\color{#7eb8a4}{\cos(2t)}}}{4}\cdot{\color{#7eb8a4}{\sin(2t)}} \end{aligned}\]
The \(\tfrac{\sin(2t)\cos(2t)}{4}\) terms cancel:
\[ y_p = -\frac{{\color{#e8b86d}{\cos(2t)}}}{4}\ln|\sec(2t)+\tan(2t)| \]
General solution
\[ y = c_1\cos(2t) + c_2\sin(2t) - \frac{\cos(2t)}{4}\ln\bigl|\sec(2t)+\tan(2t)\bigr| \]
Lecture · B4.6.7 \(y'' + 4y' + 4y = e^{-2t}\ln t\) VOP with \(\ln t\), integration by parts ▾
1
Complementary solution — double root
\[ r^2+4r+4=(r+2)^2=0 \;\Rightarrow\; r=-2,\,-2 \]
\[ y_c = c_1{\color{#e8b86d}{e^{-2t}}} + c_2{\color{#7eb8a4}{te^{-2t}}} \;\Rightarrow\; {\color{#e8b86d}{y_1=e^{-2t}}},\;{\color{#7eb8a4}{y_2=te^{-2t}}} \]
2
Wronskian
\[\begin{aligned} {\color{#e8b86d}{y_1'}} &= -2{\color{#e8b86d}{e^{-2t}}}\\[3pt] {\color{#7eb8a4}{y_2'}} &= (1-2t){\color{#7eb8a4}{e^{-2t}}} \end{aligned}\]
\[ {\color{#8b9eca}{W}} = {\color{#e8b86d}{e^{-2t}}}(1-2t){\color{#7eb8a4}{e^{-2t}}} - (-2{\color{#e8b86d}{e^{-2t}}})\cdot t{\color{#7eb8a4}{e^{-2t}}} = e^{-4t}(1-2t+2t) = {\color{#8b9eca}{e^{-4t}}} \]
3
Compute \(u_1'\) and \(u_2'\) — exponentials cancel
\[ u_1' = \frac{-{\color{#7eb8a4}{te^{-2t}}}\cdot e^{-2t}\ln t}{{\color{#8b9eca}{e^{-4t}}}} = \frac{-t\,\bcancel{e^{-4t}}\ln t}{\bcancel{e^{-4t}}} = -t\ln t \]
\[ u_2' = \frac{{\color{#e8b86d}{e^{-2t}}}\cdot e^{-2t}\ln t}{{\color{#8b9eca}{e^{-4t}}}} = \frac{\bcancel{e^{-4t}}\ln t}{\bcancel{e^{-4t}}} = \ln t \]
The \(e^{-4t}\) in the Wronskian cancels perfectly with the \(e^{-2t}\) factors. This is why the problem was designed with \(e^{-2t}\) — it matches the \(y_1, y_2\) structure.
4
Integrate \(u_1'\) by parts: \(u = \ln t,\; dv = t\,dt\)
\[ u_1 = -\int t\ln t\,dt = -\left[\frac{t^2}{2}\ln t - \int\frac{t}{2}\,dt\right] = -\frac{t^2\ln t}{2} + \frac{t^2}{4} \]
5
Integrate \(u_2'\) by parts: \(u = \ln t,\; dv = dt\)
\[ u_2 = \int \ln t\,dt = t\ln t - t \]
6
Assemble \(y_p = u_1{\color{#e8b86d}{y_1}} + u_2{\color{#7eb8a4}{y_2}}\) — watch the \(\ln t\) terms combine
\[\begin{aligned} y_p &= \Bigl(-\frac{t^2\ln t}{2}+\frac{t^2}{4}\Bigr){\color{#e8b86d}{e^{-2t}}} + (t\ln t - t)\cdot{\color{#7eb8a4}{te^{-2t}}}\\[6pt] &= e^{-2t}\Bigl(-\tfrac{t^2\ln t}{2}+\tfrac{t^2}{4}+t^2\ln t - t^2\Bigr)\\[6pt] &= e^{-2t}\Bigl(\frac{t^2\ln t}{2} - \frac{3t^2}{4}\Bigr) \end{aligned}\]
General solution
\[ y = c_1 e^{-2t} + c_2 t e^{-2t} + e^{-2t}\!\left(\frac{t^2\ln t}{2} - \frac{3t^2}{4}\right) \]

Original problems — none from the slides. Each one is designed to isolate a specific concept.

New · 1 \(y'' - 3y' + 2y = 4e^{3t}\) Exponential — no overlap ▾
1
Complementary solution
\[ r^2-3r+2=(r-1)(r-2)=0 \;\Rightarrow\; r=1,\,2 \;\Rightarrow\; y_c=c_1 e^{t}+c_2 e^{2t} \]
\(f(t)=4e^{3t}\). Is \(e^{3t}\) in \(y_c\)? No — roots are 1 and 2, not 3. No overlap.
2
Guess and differentiate
\[ y_p = {\color{#e8b86d}{A}}e^{3t},\quad y_p' = 3{\color{#e8b86d}{A}}e^{3t},\quad y_p'' = 9{\color{#e8b86d}{A}}e^{3t} \]
3
Substitute and solve
\[ 9{\color{#e8b86d}{A}}e^{3t} - 3(3{\color{#e8b86d}{A}}e^{3t}) + 2({\color{#e8b86d}{A}}e^{3t}) = 4e^{3t} \]
\[ {\color{#e8b86d}{A}}(9-9+2)\,e^{3t} = 4e^{3t} \;\Rightarrow\; 2{\color{#e8b86d}{A}}=4 \;\Rightarrow\; {\color{#e8b86d}{A=2}} \]
General solution
\[ y = c_1 e^t + c_2 e^{2t} + 2e^{3t} \]
New · 2 \(y'' - 3y' + 2y = 5e^{t}\) Same ODE — now single overlap ▾
1
Same \(y_c\) — but now \(e^t\) is a root
\[ y_c = {\color{#c47b7b}{c_1 e^{t}}} + c_2 e^{2t} \]
Normal guess \({\color{#c47b7b}{Ae^t}}\) is in \(y_c\). Single overlap → multiply by \(t\).
2
Correct guess and derivatives
\[\begin{aligned} y_p &= {\color{#e8b86d}{A}}te^t\\[4pt] y_p' &= {\color{#e8b86d}{A}}(t+1)e^t\\[4pt] y_p'' &= {\color{#e8b86d}{A}}(t+2)e^t \end{aligned}\]
3
Substitute — \(t\) terms cancel as guaranteed

General solution

New · 3 \(y'' + 2y' = 6t + 4\) Polynomial overlap via \(r=0\) ▾
1
Complementary solution — \(r=0\) is a root
\[ r^2 + 2r = r(r+2) = 0 \;\Rightarrow\; r=0,\,-2 \;\Rightarrow\; y_c = {\color{#c47b7b}{c_1}} + c_2 e^{-2t} \]
⚠ \(r=0\) means constants are solutions to the homogeneous equation. The usual guess for a degree-1 polynomial is \(At+B\), but \({\color{#c47b7b}{B}}\) (a constant) is in \(y_c\). Overlap → multiply by \(t\).

Correct guess

\[ 2{\color{#e8b86d}{A}} + 2(2{\color{#e8b86d}{A}}t+{\color{#7eb8a4}{B}}) = 6t+4 \;\Rightarrow\; 4{\color{#e8b86d}{A}}t + (2{\color{#e8b86d}{A}}+2{\color{#7eb8a4}{B}}) = 6t+4 \]

General solution
\[ y = c_1 + c_2 e^{-2t} + \tfrac{3}{2}t^2 + \tfrac{1}{2}t \]
New · 4 \(y'' + y = \sec t \tan t\) VOP — not MUC viable ▾
1
Setup — \(a=1\), find \(y_1,y_2,W\)
\[ r^2+1=0 \;\Rightarrow\; {\color{#e8b86d}{y_1=\cos t}},\;{\color{#7eb8a4}{y_2=\sin t}},\;{\color{#8b9eca}{W=1}} \]
2
\(u_1'\) and \(u_2'\)
\[ u_1' = \frac{-{\color{#7eb8a4}{\sin t}}\cdot\sec t\tan t}{{\color{#8b9eca}{1}}} = -\tan^2 t = -({\sec^2 t - 1}) \]
\[ u_2' = \frac{{\color{#e8b86d}{\cos t}}\cdot\sec t\tan t}{{\color{#8b9eca}{1}}} = \tan t \]
3
Integrate
\[ u_1 = -\int(\sec^2 t - 1)\,dt = -\tan t + t \]
\[ u_2 = \int\tan t\,dt = \ln|\sec t| \]
4
Assemble \(y_p = u_1{\color{#e8b86d}{y_1}} + u_2{\color{#7eb8a4}{y_2}}\)
\[\begin{aligned} y_p &= (-\tan t + t){\color{#e8b86d}{\cos t}} + \ln|\sec t|\cdot{\color{#7eb8a4}{\sin t}}\\[4pt] &= -\sin t + t\cos t + \sin t\ln|\sec t| \end{aligned}\]
The \(-\sin t\) absorbs into \(c_2\sin t\).
General solution
\[ y = c_1\cos t + c_2\sin t + t\cos t + \sin t\ln|\sec t| \]
